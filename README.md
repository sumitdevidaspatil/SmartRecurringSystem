# 🚀 RD System

## Overview

RD System is a Full Stack Web Application built using React.js, Spring Boot, Hibernate, and PostgreSQL.

## Features

- User Registration & Login
- RD Account Management
- Dashboard Analytics
- Passbook Management
- Loan Management
- EMI Calculator
- Reports Dashboard
